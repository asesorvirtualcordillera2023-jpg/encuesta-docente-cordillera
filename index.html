Gestión de formularios dinámicos

Objetivo: crear diferentes encuestas o seguimientos institucionales. Cada formulario puede tener sus propias categorías, preguntas, público objetivo y reportes.

Categorías por formulario

Organiza las preguntas por secciones. Ejemplo: Aula virtual, desempeño docente, tutoría, satisfacción, seguimiento académico.

Activación por periodo y modalidad



Gestión de preguntas

Flujo recomendado: 1) crea el formulario, 2) crea sus categorías, 3) crea o importa las preguntas.

Crear / editar pregunta individual



Importar preguntas por archivo

Columnas recomendadas: formulario, categoria, orden, pregunta, tipo, obligatoria, estado. Para XML Moodle Feedback se detectan las etiquetas de sección y las preguntas automáticamente.

Ingreso de nuevos administradores con roles

JPG/PNG hasta 2 MB. Se guarda como URL/base64.

Puedes ajustar los módulos de forma manual aunque el rol sugiera una configuración inicial.


Registros de encuestas recibidas

Este módulo es independiente de reportes. Permite revisar y eliminar respuestas llenadas, por ejemplo registros de prueba o duplicados.

Subida de base de docentes en CSV

Carga la base de docentes de cada semestre. El archivo debe incluir columnas como: MODALIDAD, JORNADA, CARRERA, NIVEL, PAR, MATERIA, PROFESOR, PERIODO. La carga actualiza catálogos y relaciones materia-docente-periodo sin borrar respuestas históricas.

Base de estudiantes

Registra estudiantes para formularios o procesos que sí requieren seguimiento individual, como tutorías de titulación, prácticas, vinculación o acompañamiento académico.

Tutorías y seguimientos académicos

Vincula estudiantes con docentes tutores y formularios específicos. Útil para titulación, proyectos integradores, prácticas, vinculación o seguimiento académico.
